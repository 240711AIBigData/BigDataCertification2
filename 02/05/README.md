# CAHPTER05 머신러닝 실습(다중 분류)
- 이진 분류(Binary Classification) : 0 또는 1, 불량 또는 정상 등 클래스(class)가 2개

- 다중 분류(Multi-class Classification) : 클래스가 3개 이상

  - 평가지표에만 차이 有
 
- LightGBM 활용해 범주형 변수를 인코딩 없이 전처리
